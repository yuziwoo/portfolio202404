## 🤙🏻 커밋 컨벤션

```

```

- feat: 새로운 기능 구현
- fix: 버그 수정
- chore: 기타 업데이트
- ci: CI/CD 파이프라인 변경
- comment: 주석 업데이트
- docs: 문서 수정
- design: 디자인, UI 변경
- refactor: 코드 리팩터링 (기능 수정 ❌)
- style: 코드 포맷, 세미 콜론 누락 수정 (코드 수정 ❌)
